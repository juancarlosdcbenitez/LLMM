5 propiedades CSS útiles

  • Color
  • Background
  • Margin
  • Padding
  • Border

1. color


Define el color del texto dentro de un elemento.


p { color: red; }

Referencia W3C



2. background


Permite establecer un color o imagen de fondo para un elemento.


div { background: yellow; }

Referencia W3C



3. margin


Define el espacio exterior alrededor de un elemento.


h1 { margin: 20px; }

Referencia W3C



4. padding


Define el espacio interior entre el contenido y el borde del elemento.


div { padding: 10px; }

Referencia W3C



5. border


Permite definir el estilo, el ancho y el color del borde de un elemento.


p { border: 1px solid black; }

Referencia W3C

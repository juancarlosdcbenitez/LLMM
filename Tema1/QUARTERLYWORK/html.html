5 useful HTML tags


  • Header
  • Nav
  • Section
  • Article
  • Footer


1. <header>


Defines the header of a page or section, usually with a logo and navigation.


<header>Welcome to Web4U</header>

W3C Reference



2. <nav>


Contains the main navigation links of the page.


<nav><a href="index.html">Home</a></nav>

W3C Reference



3. <section>


Defines a thematic section within the document.


<section><h2>News</h2></section>

W3C Reference



4. <article>


Represents independent, self-contained content, such as an article or blog post.


<article><h2>Post</h2><p>Content...</p></article>

W3C Reference



5. <footer>


Defines the footer of a section or document, with additional information or credits.


<footer>© 2025 Web4U</footer>

W3C Reference
